# Smart Weather

## Overview
This app displays the 5 day weather forecast for London city. The App also displays minimum and maximum temperature, weather conditions such as cloudy, rain, sunny etc. of current city. Swipe up to see all 5 days weather forecast of current city.

## Improvement
* Change city option
* Hourly forecast of current city
